## Project Overview
This project demonstrates the application of advanced SQL queries including Common Table Expressions (CTEs) and various Window Functions (Ranking & Navigation) using an Oracle Database environment for a Supermarket Management System.

## Database Schema
The system consists of three core tables:
1. **Products**: Contains product details, pricing, and stock levels.
2. **Sales**: Contains transaction dates and total invoice amounts.
3. **SaleDetails**: A bridge table containing itemized invoice details.
